### 2번째 프로젝트 Text RPG입니다
진행 방식은 캐릭터 생성 후 스펙업을 해서 던전을 클리어하는 것입니다.</br>
던전의 난이도는 총 3개입니다. </br>
각 던전에서 몬스터를 토벌할 시 경험치와 골드를 획득할 수 있고, 확률적으로 아이템도 드랍됩니다.</br>
3단계 던전의 보스몬스터를 토벌할 시 한정 아이템을 획득할 수 있습니다.</br>
